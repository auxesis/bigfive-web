# bigfive-web

**Do the test now at [bigfive.fractio.nl](https://bigfive.fractio.nl/).**

Website for five factor model of personality ([OCEAN](https://en.wikipedia.org/wiki/Big_Five_personality_traits)) based on work from [IPIP-NEO-PI](https://github.com/kholia/IPIP-NEO-PI).

Tests and evaluation is gathered from [ipip.ori.org](http://ipip.ori.org).

## Help wanted

If you want to help by translating the items to other languages look [here](https://b5.translations.alheimsins.net/).
